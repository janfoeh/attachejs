new Attache(anchorElement, options, callbacks)

Attache
Parameters:
Name Type Argument Description
anchorElement jQuery The DOM element Attaché will be anchored to
options Object <optional>
Properties
Name Type Argument Default Description
trigger String <optional>
 hover An event type to trigger Attaché. Allowed are "hover", "click" or "none"
position String <optional>
 right center the popover position, relative to the anchor: left|center|right top|center|bottom
alternativePositions= String[] <optional>
 an array list of alternative position strings, in case the default position leads to the popover being off-screen
offsetX Number <optional>
 10 the horizontal distance between anchor and popover in px
offsetY Number <optional>
 10 the vertical distance between anchor and popover in px
popoverClass String <optional>
 additional CSS class(es) to apply to the popover markup
debug Boolean <optional>
 false provide debug information and error handling in the console
callbacks Object <optional>
 callbacks to be triggered at various lifecycle moments
Properties
Name Type Argument Description
afterCreate afterCreateCallback <optional>
 called after the popover markup has been created
afterShow afterShowCallback <optional>
 called each time after the popover is shown
Source:

Examples
var popover = new Attache(document.getElementById('popover-link'), {popoverClass: 'info-tooltip'});
// show a tooltip to the bottom right of #popover-link, with fallback positions 'right center'
// or 'left center' if the tooltip cannot fit at bottom right
var popover = new Attache($('#popover-link'), {position: 'right bottom', alternativePositions: ['right center', 'left center']});
